Политика обработки персональных данных

1. Общие положения

Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с применимым законодательством о защите персональных данных и определяет порядок обработки и защиты информации о физических лицах (далее — «Пользователи»), приобретающих неисключительную лицензию на программное обеспечение Cyber Security PRO (далее — «ПО»).

Политика действует в отношении всех данных, которые Правообладатель ПО может получить о Пользователе во время использования сайта и приобретения лицензии. Использование сайта и приобретение лицензии означает безоговорочное согласие Пользователя с условиями настоящей Политики. В случае несогласия Пользователь должен воздержаться от использования сайта.

2. Основные термины

Персональные данные — любая информация, относящаяся к прямо или косвенно определённому или определяемому физическому лицу (субъекту персональных данных).

Обработка персональных данных — любое действие или совокупность действий с персональными данными, включая сбор, запись, систематизацию, накопление, хранение, уточнение, извлечение, использование, передачу, обезличивание, блокирование, удаление и уничтожение.

Оператор — Правообладатель программного обеспечения Cyber Security PRO, организующий обработку персональных данных, определяющий цели обработки, состав обрабатываемых данных и действия с ними.

Пользователь — физическое лицо, приобретающее неисключительную лицензию на использование ПО и предоставляющее свои персональные данные при оформлении платежа.

3. Какие данные обрабатываются

При приобретении лицензии Оператор получает от платёжной системы следующие категории данных:

  • Адрес электронной почты — указанный Пользователем при оформлении платежа. Используется исключительно для отправки лицензионного ключа и уведомлений, связанных с действием лицензии.
  • Информация о платеже — факт осуществления платежа, сумма, идентификатор транзакции в платёжной системе. Используется для верификации лицензии и бухгалтерской отчётности.
  • Доменное имя — адрес сайта, на котором активируется лицензионный ключ. Сохраняется для предотвращения несанкционированного использования лицензии на других сайтах.

Оператор не собирает, не хранит и не обрабатывает: пароли, полные номера банковских карт, CVV/CVC-коды, платёжные токены, данные для входа в интернет-банк. Обработка платёжной информации осуществляется исключительно сертифицированными платёжными системами, соответствующими международным стандартам безопасности.

Оператор не осуществляет сбор биометрических данных, а также данных о расовой или национальной принадлежности, политических взглядах, религиозных или философских убеждениях, состоянии здоровья.

4. Цели обработки данных

Персональные данные Пользователя обрабатываются исключительно в следующих целях:

  1. Предоставление лицензионного ключа после успешной оплаты — отправка ключа на указанный адрес электронной почты.
  2. Верификация лицензии при активации и последующих проверках — подтверждение действительности лицензионного ключа.
  3. Исполнение обязательств по публичной оферте — выполнение условий лицензионного соглашения.
  4. Обеспечение технической поддержки — ответы на запросы Пользователя, помощь в активации и использовании ПО.
  5. Предотвращение мошенничества и несанкционированного использования лицензий.

Персональные данные не используются в маркетинговых, рекламных или иных целях, не связанных напрямую с предоставлением лицензии и технической поддержкой.

5. Передача данных третьим лицам

Оператор не передаёт персональные данные Пользователей третьим лицам, за исключением следующих случаев:

  • Платёжным системам — в объёме, необходимом для осуществления платежа. Платёжные системы являются самостоятельными операторами персональных данных.
  • Государственным органам — в случаях, предусмотренных применимым законодательством, по официальному запросу уполномоченных органов.

6. Хранение и защита данных

Персональные данные Пользователей хранятся на защищённых серверах. Оператор принимает необходимые организационные и технические меры для защиты данных от неправомерного доступа, уничтожения, изменения, блокирования, копирования, распространения и иных неправомерных действий.

Меры защиты включают: использование защищённого протокола HTTPS, ограничение доступа к данным только уполномоченным лицам, регулярное обновление программного обеспечения серверов, резервное копирование данных.

Срок хранения персональных данных: в течение срока действия лицензии Пользователя, а также в течение 3 (трёх) лет после его окончания для целей бухгалтерской и налоговой отчётности, если иной срок не установлен законодательством.

7. Права Пользователя

Пользователь имеет следующие права в отношении своих персональных данных:

  • Право на доступ — запросить информацию об обрабатываемых данных.
  • Право на исправление — потребовать уточнения неточных данных.
  • Право на удаление — потребовать удаления данных, если это не противоречит законодательству.
  • Право на отзыв согласия — отозвать согласие на обработку данных в любое время.

Для реализации своих прав Пользователь может направить запрос через страницу контактов. Запрос будет обработан в течение 30 календарных дней.

8. Файлы cookie

Сайт использует исключительно технические файлы cookie (localStorage браузера) для сохранения выбранного языка интерфейса. Маркетинговые, аналитические и отслеживающие файлы cookie не используются.

9. Заключительные положения

Оператор оставляет за собой право вносить изменения в настоящую Политику. Новая редакция вступает в силу с момента её публикации на сайте. Продолжение использования сайта и лицензии после изменений означает согласие Пользователя с новой редакцией.

Настоящая Политика является публичным документом, доступным для ознакомления неограниченному кругу лиц.
